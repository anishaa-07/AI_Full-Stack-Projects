## 📖 Overview

CoverCraft AI is a modern cover letter generation platform designed to help students, developers, and professionals create professional cover letters quickly and efficiently.

The project features a premium glassmorphism UI, interactive form inputs, live preview generation, and a clean user experience.

---

✅ Premium Landing Page
