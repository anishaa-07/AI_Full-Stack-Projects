# 🤖 AI Code Reviewer

An AI-powered code review tool that analyzes code snippets and provides intelligent feedback on bugs, improvements, optimizations, and best practices using Google's Gemini AI.

## 🚀 Features

* 🔍 AI-powered code analysis
* 🐞 Detects bugs and potential issues
* ✨ Suggests code improvements
* ⚡ Recommends performance optimizations
* 📖 Highlights best practices
* 🎨 Modern Wine-Themed UI
* 🤖 Powered by Gemini AI

## 🛠️ Tech Stack

* React.js
* Vite
* JavaScript
* Gemini AI API
* CSS3

## 📸 Preview

Paste your code into the editor and get instant AI feedback.

Example:

```javascript
function add(a,b){
return a+b
}
```

The AI reviews:

* Code quality
* Formatting issues
* Best practices
* Performance suggestions

## ⚙️ Installation

1. Clone the repository

```bash
git clone <repository-url>
```

2. Navigate to project

```bash
cd AI-Code-Reviewer
```

3. Install dependencies

```bash
npm install
```

4. Add Gemini API Key

Create a `.env` file:

```env
VITE_GEMINI_API_KEY=YOUR_API_KEY
```

5. Start development server

```bash
npm run dev
```
