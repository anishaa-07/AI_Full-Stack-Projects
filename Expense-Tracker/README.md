# 💰 Expense Tracker

### Track Every Transaction. Take Control Of Your Finances.

A modern Expense Tracker built with React that allows users to manage their income and expenses efficiently. The application provides real-time balance calculations, transaction history management, and persistent data storage using LocalStorage.

---

## 🚀 Features

- Add Income Transactions
- Add Expense Transactions
- Real-Time Balance Calculation
- Transaction History
- Delete Transactions
- LocalStorage Data Persistence
- Responsive User Interface
- Fast & Lightweight Performance

---

## 🛠️ Tech Stack

- JavaScript (ES6+)
- CSS3
- LocalStorage API
- Vite

## ⚙️ How It Works

1. Enter a transaction description.
2. Add the transaction amount.
3. Positive value → Income
4. Negative value → Expense
5. Click Add Transaction.
6. Balance updates automatically.
7. Transactions are stored in LocalStorage.
